TSC -> É O COMPILADOR DE TYPESCRIPT

commands:

yarn tsc --init : vamos criar um arquivos de configuração para typescript.

yarn tsc : vai pegar o arquivos tsconfig.json criado pelo command acima e compilar o ts para js.
